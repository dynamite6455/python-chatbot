# python-chatbot
A slow AI chatbot built directly into Python. You can access this project through any sort of Python IDE such as Thonny!

This code uses the TIME and REQUESTS modules. The REQUESTS module actually requires a package from PyPI. You can go to Tools > Manage packages... > Search and search for "requests". Without the REQUESTS package, the code will not recognize the module. With the module, you can import an ML model to use.
